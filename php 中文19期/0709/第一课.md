# http 基础

- http：超文本传输协议，类似于合同，双方都要遵守的约定
- 超文本：使用标签描述内容，并且由层级结构
- 特点：无状态，每次请求对服务器来说都是全新的，尽管请求来自同一个客户端
- URL：用来标注资源，像是完整的地址，包含你详细的住址，如何到达等
- URI：用来描述 URI 资源的字符串，就像一个身份证，是一个唯一的标识
- DOM：使用文档对象模型描述 html
